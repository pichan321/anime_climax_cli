# anime_climax_cli

```
Anime Climax CLI 1.0
Pichsereyvattana Chan
A Rust CLI to help you get your favorite anime climax scene in seconds

USAGE:
    anime_climax_cli [OPTIONS]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -c, --clips <clips> list clips of an anime with specified anime id
    -d, --download_all <download_all_clips> of an anime with specified anime id
    -p, --page <page> Page number (p >= 1)
    -s, --search <search> Anime name to search
        --sc <single_clip> Download clip with specified (id)
```
